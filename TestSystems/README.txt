This folder contains the test systems provided with MultiUbic v.0.3

The folder Dijkstra contains two subfolders called MultiUbic and UBIC2.
In this folder there are 45 two-level net systems modeling  the famous problem of the dining philosophers.
The numer of philosophers appears in the file name.
The MultiUbic subfolder additionally contains a file called LH.mds, modelling the two level policy.

The folder T has the same substructure and is meant to be populated by the testNetGen tool.
